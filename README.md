# AVOTS CIC Website

Static website for AVOTS CIC — Research. Education. Community.

## Publish with GitHub Pages

1. Create a new repository on GitHub.
2. Upload all files and the complete `assets` folder from this package to the repository root.
3. Open the repository's **Settings**.
4. Select **Pages** under **Code and automation**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the **main** branch and **/(root)** folder, then click **Save**.

GitHub will display the public website address after deployment completes.

## Website files

- `index.html` — main one-page website
- `project-summer-camp.html` — dedicated summer camp project page
- `styles.css` and `project-page.css` — visual design
- `script.js` and `project-page.js` — navigation, animations and contact interaction
- `assets/` — website images and favicon

The contact form opens the visitor's email application. No server or database is required.
